February 2013 CBSA Delineation - Geospatial Files

These shapefiles and geodatabases represent the February 2013 Core based statistical areas (CBSA) delineations released by the Office of Management & Budget.

The files are an interim product until the 2013 TIGER/Line Shapefiles are released in late summer/early fall 2013.  These files use the 2012 TIGER/Line Shapefiles county boundaries (http://www2.census.gov/geo/tiger/TIGER2012/COUNTY/tl_2012_us_county.zip) and the CBSA delineation file (http://www.census.gov/population/metro/files/lists/2013/List1.xls).

The geodatabases were created using ArcGIS 10.1.

Four layers are available:
- Counties
- CBSAs
- Combined statistical area (CSAs)
- Metropolitan division (MetDiv)

Counties:
STATEFP - State FIPS code
COUNTYFP - County FIPS code
COUNTYNS - County GNIS code
GEOID - 5-digit State & County code
NAME - County Name
NAMELSAD - Full county name
LSAD - Legal/Statistical Area Descriptor
CLASSFP - County Class Code
MTFCC - MAF/TIGER Feature Classification Code
FUNCSTAT - Functional Status
ALAND - Land Area (square meters)
AWATER - Water Area (square meters)
INTPLAT - Internal Point Latitude Coordinate
INTPTLON - Internal Point Longitude Coordinate
CBSAFP - 5-digit CBSA code
MDIVFP - 5-digit Metropolitan division code
CSAFP - 3-digit CSA code
M_Status - Metropolitan or Micropolitan statistical area
MDIV_Title - name of Metropolitan division
CSA_Title - name of CSA
C_Status - Central County or Outlying County


CBSAs:
CBSAFP - 5-digit CBSA code
CBSA_Title - name of CBSA
M_Status - Metropolitan or Micropolitan statistical area

CSAs:
CSAFP - 3-digit CSA code
CSA_Title - name of CSA

MetDiv:
METDIVFP - 5-digit Metropolitan division code
MDIV_Title - name of Metropolitan division

For more information about the metropolitan and micropolitan statistical areas as of February 2013 please see the Office of Management and Budget (OMB) Bulletin No. 13-01 (http://www.whitehouse.gov/sites/default/files/omb/bulletins/2013/b-13-01.pdf).  Questions about the definition of metropolitan and micropolitan statistical areas should be directed to the Office of Mangement and Budget.
